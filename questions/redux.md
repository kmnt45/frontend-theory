# Redux

## 1. –ß—Ç–æ —Ç–∞–∫–æ–µ Flux?

`Flux` ‚Äî —ç—Ç–æ –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω, –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω—ã–π Facebook –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º –≤ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è—Ö.

–ì–ª–∞–≤–Ω–∞—è –∏–¥–µ—è ‚Äî –æ–¥–Ω–æ–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–π –ø–æ—Ç–æ–∫ –¥–∞–Ω–Ω—ã—Ö, —á—Ç–æ —Å–Ω–∏–∂–∞–µ—Ç —Å–ª–æ–∂–Ω–æ—Å—Ç—å –±–æ–ª—å—à–∏—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π.

üîë –û—Å–Ω–æ–≤–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã:

- `Actions` ‚Äî –æ–ø–∏—Å—ã–≤–∞—é—Ç, —á—Ç–æ –ø—Ä–æ–∏–∑–æ—à–ª–æ (—Ç–∏–ø + –¥–∞–Ω–Ω—ã–µ).

- `Dispatcher` ‚Äî —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—ã–π —Ö–∞–±, –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç actions –≤ stores.

- `Stores` ‚Äî —Ö—Ä–∞–Ω–∏–ª–∏—â–∞ —Å–æ—Å—Ç–æ—è–Ω–∏—è + –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–∞.

- `Views` ‚Äî UI-–∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã, –ø–æ–¥–ø–∏—Å–∞–Ω–Ω—ã–µ –Ω–∞ stores.

üí° –ù–∞ —Å–æ–±–µ—Å–µ: –º–æ–∂–Ω–æ —Å–∫–∞–∑–∞—Ç—å, —á—Ç–æ Redux ‚Äî —ç—Ç–æ –±–æ–ª–µ–µ —Å—Ç—Ä–æ–≥–∞—è –∏ —É–ø—Ä–æ—â—ë–Ω–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –∏–¥–µ–π Flux, –≥–¥–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ö—Ä–∞–Ω–∏—Ç—Å—è —Ü–µ–Ω—Ç—Ä–∞–ª–∏–∑–æ–≤–∞–Ω–Ω–æ –∏ –º–µ–Ω—è–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ —á–µ—Ä–µ–∑ reducer.

## 2. –ß—Ç–æ —Ç–∞–∫–æ–µ Redux? –ö–ª—é—á–µ–≤—ã–µ –ø—Ä–∏–Ω—Ü–∏–ø—ã Redux?

`Redux` ‚Äî —ç—Ç–æ –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π, –≤–¥–æ—Ö–Ω–æ–≤–ª—ë–Ω–Ω–∞—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–æ–π Flux.
–û–Ω–∞ –æ—Å–æ–±–µ–Ω–Ω–æ –ø–æ–ª–µ–∑–Ω–∞ –≤ –±–æ–ª—å—à–∏—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è—Ö —Å –º–Ω–æ–∂–µ—Å—Ç–≤–æ–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤, –≥–¥–µ –Ω—É–∂–Ω–æ —Ü–µ–Ω—Ç—Ä–∞–ª–∏–∑–æ–≤–∞–Ω–Ω–æ–µ –∏ –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º–æ–µ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º.

üîë –ö–ª—é—á–µ–≤—ã–µ –ø—Ä–∏–Ω—Ü–∏–ø—ã Redux:

1. –ï–¥–∏–Ω—ã–π –∏—Å—Ç–æ—á–Ω–∏–∫ –∏—Å—Ç–∏–Ω—ã (Single Source of Truth)
–í—Å—ë —Å–æ—Å—Ç–æ—è–Ω–∏–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ –æ–¥–Ω–æ–º –æ–±—ä–µ–∫—Ç–µ (state tree), –≤–Ω—É—Ç—Ä–∏ –æ–¥–Ω–æ–≥–æ –≥–ª–æ–±–∞–ª—å–Ω–æ–≥–æ store.

2. –°–æ—Å—Ç–æ—è–Ω–∏–µ —Ç–æ–ª—å–∫–æ –¥–ª—è —á—Ç–µ–Ω–∏—è.
–ò–∑–º–µ–Ω–∏—Ç—å —Å–æ—Å—Ç–æ—è–Ω–∏–µ –Ω–∞–ø—Ä—è–º—É—é –Ω–µ–ª—å–∑—è. –ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π —Å–ø–æ—Å–æ–± –∏–Ω–∏—Ü–∏–∏—Ä–æ–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏–µ ‚Äî —ç—Ç–æ –æ—Ç–ø—Ä–∞–≤–∏—Ç—å action (–æ–±—ã—á–Ω—ã–π –æ–±—ä–µ–∫—Ç —Å –ø–æ–ª–µ–º type).

3. –ò–∑–º–µ–Ω–µ–Ω–∏—è –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è —á–∏—Å—Ç—ã–º–∏ —Ñ—É–Ω–∫—Ü–∏—è–º–∏ (reducers).
–†–µ–¥—å—é—Å–µ—Ä—ã –ø—Ä–∏–Ω–∏–º–∞—é—Ç —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏ action, –≤–æ–∑–≤—Ä–∞—â–∞—è –Ω–æ–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ.
–û–Ω–∏ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —á–∏—Å—Ç—ã–º–∏ —Ñ—É–Ω–∫—Ü–∏—è–º–∏: –±–µ–∑ –ø–æ–±–æ—á–Ω—ã—Ö —ç—Ñ—Ñ–µ–∫—Ç–æ–≤ –∏ –º—É—Ç–∞—Ü–∏–π, –≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

_Redux —Ö–æ—Ä–æ—à –¥–ª—è –∫—Ä—É–ø–Ω—ã—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π —Å –æ–±—â–∏–º —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º, –¥–µ–ª–∞–µ—Ç –µ–≥–æ –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–º –∏ –ª–µ–≥–∫–æ —Ç–µ—Å—Ç–∏—Ä—É–µ–º—ã–º, –Ω–æ –≤ –º–∞–ª–µ–Ω—å–∫–∏—Ö –ø—Ä–æ–µ–∫—Ç–∞—Ö –º–æ–∂–µ—Ç –±—ã—Ç—å –ª–∏—à–Ω–∏–º –∏ –¥–æ–±–∞–≤–ª—è—Ç—å boilerplate._

## 3. –ß—Ç–æ —Ç–∞–∫–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ (Action)?

`–î–µ–π—Å—Ç–≤–∏–µ (Action)` ‚Äî —ç—Ç–æ –æ–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π –æ–ø–∏—Å—ã–≤–∞–µ—Ç –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è –≤ Redux. –û–±—è–∑–∞—Ç–µ–ª—å–Ω–æ–µ —Å–≤–æ–π—Å—Ç–≤–æ –¥–µ–π—Å—Ç–≤–∏—è ‚Äî —ç—Ç–æ `type`, —É–∫–∞–∑—ã–≤–∞—é—â–µ–µ –Ω–∞ —Ç–∏–ø –∏–∑–º–µ–Ω–µ–Ω–∏—è. –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è —á–µ—Ä–µ–∑ `payload`.

```js
{
  type: 'ADD_TODO',
  payload: { text: '–ö—É–ø–∏—Ç—å –º–æ–ª–æ–∫–æ' }
}
```

_`Action creator` ‚Äî —ç—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç action. –í —Å–≤—è–∑–∫–µ —Å middleware (–Ω–∞–ø—Ä–∏–º–µ—Ä, Thunk) –æ–Ω–∏ –º–æ–≥—É—Ç –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å –Ω–µ —Ç–æ–ª—å–∫–æ –æ–±—ä–µ–∫—Ç, –Ω–æ –∏ —Ñ—É–Ω–∫—Ü–∏—é –¥–ª—è –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π._

```ts
// Action
const increment = { type: "INCREMENT" };

// Action creator
function incrementAction() {
  return { type: "INCREMENT" };
}

function addTodoAction(text: string) {
  return { type: "ADD_TODO", payload: text };
}

```

## 4. –ß—Ç–æ —Ç–∞–∫–æ–µ –¥–∏—Å–ø–µ—Ç—á–µ—Ä (Dispatch)?

`–î–∏—Å–ø–µ—Ç—á–µ—Ä (Dispatch)` ‚Äî —ç—Ç–æ –º–µ—Ç–æ–¥, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ –¥–µ–π—Å—Ç–≤–∏–π (actions) –≤ —Ö—Ä–∞–Ω–∏–ª–∏—â–µ (store) Redux. –ö–æ–≥–¥–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è dispatch(action), —Ö—Ä–∞–Ω–∏–ª–∏—â–µ –ø–µ—Ä–µ–¥–∞—ë—Ç —ç—Ç–æ –¥–µ–π—Å—Ç–≤–∏–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–º—É —Ä–µ–¥—å—é—Å–µ—Ä—É, –∫–æ—Ç–æ—Ä—ã–π –≤—ã—á–∏—Å–ª—è–µ—Ç –Ω–æ–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –Ω–∞ –æ—Å–Ω–æ–≤–µ —Ç–µ–∫—É—â–µ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –∏ —Ç–∏–ø–∞ –¥–µ–π—Å—Ç–≤–∏—è.

## 5. –ß—Ç–æ —Ç–∞–∫–æ–µ —Ä–µ–¥—å—é—Å–µ—Ä (Reducer)?

`–†–µ–¥—å—é—Å–µ—Ä (Reducer)` ‚Äî —ç—Ç–æ —á–∏—Å—Ç–∞—è —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏ –¥–µ–π—Å—Ç–≤–∏–µ (action) –≤ –∫–∞—á–µ—Å—Ç–≤–µ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ. –†–µ–¥—å—é—Å–µ—Ä –Ω–µ –∏–∑–º–µ–Ω—è–µ—Ç –∏—Å—Ö–æ–¥–Ω–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, –∞ —Å–æ–∑–¥–∞–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ–≥–æ –∫–æ–ø–∏—é —Å —É—á—ë—Ç–æ–º –∏–∑–º–µ–Ω–µ–Ω–∏–π, –æ–ø–∏—Å–∞–Ω–Ω—ã—Ö –≤ –¥–µ–π—Å—Ç–≤–∏–∏.

## 6. –ß—Ç–æ —Ç–∞–∫–æ–µ —Ö—Ä–∞–Ω–∏–ª–∏—â–µ (Store)?

`–•—Ä–∞–Ω–∏–ª–∏—â–µ (Store)` ‚Äî —ç—Ç–æ –æ–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π —Ö—Ä–∞–Ω–∏—Ç —Å–æ—Å—Ç–æ—è–Ω–∏–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è –∏ —É–ø—Ä–∞–≤–ª—è–µ—Ç –µ–≥–æ –∏–∑–º–µ–Ω–µ–Ω–∏—è–º–∏.

## 7. –ß—Ç–æ —Ç–∞–∫–æ–µ –º–∏–¥–ª–≤–µ–π—Ä (Middleware)?

`–ú–∏–¥–ª–≤–µ–π—Ä (Middleware)` ‚Äî —ç—Ç–æ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω—ã–π —Å–ª–æ–π, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ—Ö–≤–∞—Ç—ã–≤–∞—Ç—å –∏ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å –¥–µ–π—Å—Ç–≤–∏—è (actions) –ø–µ—Ä–µ–¥ —Ç–µ–º, 
–∫–∞–∫ –æ–Ω–∏ –¥–æ—Å—Ç–∏–≥–Ω—É—Ç —Ä–µ–¥—å—é—Å–µ—Ä–æ–≤ (reducers). 
–ú–∏–¥–ª–≤–µ–π—Ä –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –¥–æ–±–∞–≤–ª—è—Ç—å –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—É—é –ª–æ–≥–∏–∫—É, –Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã–º–∏ –æ–ø–µ—Ä–∞—Ü–∏—è–º–∏, 
–ª–æ–≥–∏—Ä–æ–≤–∞–Ω–∏—è –¥–µ–π—Å—Ç–≤–∏–π, –æ–±—Ä–∞–±–æ—Ç–∫–∏ –æ—à–∏–±–æ–∫ –∏ –¥—Ä—É–≥–∏—Ö –∑–∞–¥–∞—á, –Ω–µ –Ω–∞—Ä—É—à–∞—è –æ—Å–Ω–æ–≤–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã Redux.

## 8. –†–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É React State –∏ Redux State?

- React State –ª–æ–∫–∞–ª–µ–Ω –¥–ª—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞, –∂–∏–≤–µ—Ç —Ç–æ–ª—å–∫–æ –≤–Ω—É—Ç—Ä–∏ –Ω–µ–≥–æ, —Ç–æ–≥–¥–∞ –∫–∞–∫ Redux State –≥–ª–æ–±–∞–ª–µ–Ω –∏ –¥–æ—Å—Ç—É–ø–µ–Ω –≤—Å–µ–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞–º –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è.

- React State –æ–±–Ω–æ–≤–ª—è–µ—Ç—Å—è —á–µ—Ä–µ–∑ setState (–∫–ª–∞—Å—Å—ã) –∏–ª–∏ useState (—Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã), —Ç–æ–≥–¥–∞ –∫–∞–∫ Redux State –æ–±–Ω–æ–≤–ª—è–µ—Ç—Å—è —á–µ—Ä–µ–∑ dispatch(action) –∏ —Ä–µ–¥—å—é—Å–µ—Ä—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –Ω–æ–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ (–∏–º–º—É—Ç–∞–±–µ–ª—å–Ω–æ).

- React State –¥–æ—Å—Ç—É–ø–µ–Ω –Ω–∞–ø—Ä—è–º—É—é –≤ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–µ, —Ç–æ–≥–¥–∞ –∫–∞–∫ Redux State –ø–æ–ª—É—á–∞—é—Ç —á–µ—Ä–µ–∑ –ø–æ–¥–ø–∏—Å–∫—É (connect HOC) –∏–ª–∏ —Å–µ–ª–µ–∫—Ç–æ—Ä—ã (useSelector).

## 9. –ö–∞–∫ –≤—ã–≥–ª—è–¥–∏—Ç –ø–æ—Ç–æ–∫ –¥–∞–Ω–Ω—ã—Ö –≤ Redux-–ø—Ä–∏–ª–æ–∂–µ–Ω–∏–∏?
